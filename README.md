# JICA Cairo Transport Strategy Japan Training Website

Static multilingual agenda web app for the June 16-28, 2026 Japan training program under the Greater Cairo transport infrastructure strategy project.

Open `index.html` in a browser. The page supports English, Japanese, and Arabic through the language buttons in the header. The agenda is stored in `app.js`, based on `C:\Users\blawal.hussain\Downloads\カイロ本邦研修_スケジュール_260615.xlsx` as read on June 16, 2026.

Main files:

- `index.html` - page structure and content sections
- `app.js` - interactive daily agenda, search, filters, countdown, print, and email helper
- `styles.css` - responsive visual design
- `assets/japan-training-room.png` - hero background image
